Visualising The Symmetrized Bidisc

An old university project finds a home on the web thanks to github pages.

The symmetrized bidisc is the set {z1z2, z1+z2 : |z1| < 1, |z2| < 1} where z1 and z2 are complex numbers. (I think I've got that syntax right. If anyone can provide a snippet of latex let me know and I'll try a bit of javascript voodoo to make it look nice).

This definition gives in a set of pairs of complex numbers, with each pair representing 2 real numbers, so it is a four dimensional set. This project was an attempt at visualising the set.

To see it in action, go to the applet page. You'll need Java and Java3d installed. (If you're on a Mac you should be good to go.)
